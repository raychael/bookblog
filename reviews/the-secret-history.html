Title: The Secret History

Author: Donna Tartt

Rating:

Thoughts: I've been in the moood lately for some dark academia, so I had to pick up the book that paved the way for so many others in this realm. I was pretty impressed by The Secret History. I was honestly probably the most impressed with the fact it kept my attention as heavily and for as long as it did, considering how long this book really is.

This is a pretty strange read. The characters are unlikable and you're not sure if you can root for any of them half the time. But it's engaging and mysterious and absolutely perfect for spooky season.

I don't think about this book very often, and I didn't like it quite as much as I was expecting to. However, it was a fantastic read nonetheless.
